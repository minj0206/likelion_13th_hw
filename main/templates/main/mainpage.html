🦁template🦁
  1. templates 폴더 생성 - 유저에게 제공할 파일의 구조나 레이아웃을 한 데 모아 관리 위함
    1) main 앱 안에 templates 폴더 생성
    2) templates 안에 main 폴더 생성
    3) main 폴더 내 main, secondpage.html 파일 생성
  2. HTML 페이지와 view 연결 - view 작성 : 원하는 url을 통해 mainpage가 보여지게 함. (렌더링)
    def mainpage(request):
    return render(request, 'main/mainpage.html')로 작성
  3. HTML 페이지와 view 연결 - url 작성 : project > urls.py에 url에 해당하는 데이터 연결
  4. python manage.py runserver로 페이지 확인 가능
  5. HTML 링크 연결 : urls.py에서 지정한 name을 활용해 html 페이지에서 해당 페이지로 이동할 수 있도록 링크를 걸어줌
🦁template 언어🦁
: view에 작성된 data 가져오기, 반복문 등 기능 수행
  1. template 언어 : view에서 생성한 데이터는 html에서 사용 가능, for문도 활용 가능
  2. 템플릿 필터 : 템플릿 변수를 서식에 따라 변환하는 것
  3. 템플릿 주석 : html 내용 중 주석이 필요한 경우 사용 {#주석#}
